Trivia Star: A Brain-sweating Mobile Game.

  • Source Code

Alongside 3 other students, we designed and developed a mobile application that hosts a Trivia game and we called it Trivia Star.

Using modern technologies like Flutter, we established multiple quizzes in multiple categories like Science, Technology, etc.

The player can download the game, create an account, choose a quiz and start answering questions.

In the end, A summary is displayed for the player to inform them of their score and final results.

  • ✅ Implementation of multiple user-friendly interfaces using cutting-edge mobile technologies.
  • ✅ Integration Firebase as a database to store user information.
