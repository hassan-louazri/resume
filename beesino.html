BEESINO: An Online Casino Platform

  • Source Code

Working with 5 other students, we developed a web application that hosts multiple social games like Poker.

Together, we managed to establish multiple gorgeous interfaces which draw the player to the game.

We also established two different game modes for players who like to play against the computer in the solo mode or with other online players in the multiplayer mode.

Lastly we integrated paiement methods using cryptocurrency in the Ethereum blockchain which allows you to buy chips to play with.

  • ✅ Design and implementation of popular social games using cutting-edge technologies.
  • ✅ Integration of solo and multiplayer modes with different functionalities.
  • ✅ Integration of a paiement method using cryptocurrency.

Tech Stack

ReactJS, NodeJS, MongoDB, WebSockets, Figma, Blockchain
